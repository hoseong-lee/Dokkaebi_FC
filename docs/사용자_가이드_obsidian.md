## 👤 첫 접속 워크플로우

> [!important] 시작은 마이페이지에서
> 헤더 우측 **[★ 내 정보]** 클릭 → 본인 선수 연결 확인 → 프로필 사진·한마디 설정

- **연결된 선수** — 화이트리스트에 미리 매칭됐으면 자동 연결됨
- **프로필 사진** — 도깨비FC 유니폼 캐릭터 **24종** (남자 12 + 여자 12, 다양한 피부톤/헤어컬러) 또는 직접 업로드
- **한마디** — 80자 자기소개
- **내 시즌 통계** + **단짝 콤비** + **획득 뱃지** 한눈에

## 📅 [[경기]]

> [!tip] 경기 양상 배지
> 🔥 대승 (4점차+) · ✅ 승 · 🤝 무 · ❌ 패 · 💥 대패
> + 1점 차이면 🟣 **접전** 태그

**예정/완료 탭**

각 경기 상세에서:
- 📍 경기장 위치 (지도 링크)
- ✋ **RSVP** — 참석/미정/불참 실시간 토글
- 🏆 **쿼터별 스코어·득점자·포메이션** 도표
- ⭐ **MOM 투표** (참여 선수만)
- 📸 **결과 카드** — 1080×1080 인스타·단톡 공유용 자동 생성

> [!example] 4쿼터 워크플로우
> 1. 경기 등록/수정 → **스쿼드 메이커**에서 4쿼터 각각 라인업·포메이션 작성
> 2. 경기 종료 → 결과 입력에서 `📋 예정 스쿼드 가져오기` 한 번에 복사
> 3. 1쿼터 입력 → `↻ 직전 쿼터 명단 복사` 로 2/3/4쿼터 빠르게 추적

## ⚙️ 관리자 전용

### 대시보드
- 결과 미입력 알림
- 이번 주 경기
- 초기 데이터 가져오기 / 시즌 분리 / 상대팀명 정규화

### 선수 관리
- CRUD · 🎨 아바타(24종 또는 직접 업로드)
- 메인/서브 포지션 · 등번호 · ★ 고정멤버

### 스쿼드 메이커 (`/admin/matches/:id/squad` 또는 경기 등록/수정 임베드)

> [!info] 4쿼터 구조
> 각 쿼터마다 라인업·포메이션이 다를 수 있음

- ★ 고정 전체 1클릭 / `+ 용병` 즉석 등록
- ⚡ **참석 인원 기반 포메이션 자동 추천**
- 인기 포메이션 칩: `4-3-3` / `4-2-3-1` / `4-4-2` / `4-3-2-1`
- **선수별 쿼터 카운트 배지** — 1쿼만(🟠) / 2~3쿼(🟢) / 풀쿼(🟡)
- 💬 **단톡 공유 텍스트** 1클릭 복사

### 결과 입력 (4쿼터 탭)
- 📋 예정 스쿼드 가져오기 / ↻ 직전 쿼터 명단 복사
- MOM 직접 지정 또는 투표 마감

### 회비 관리
- 매치비 (경기별 출전자)
- 월회비 (월별 고정멤버)
- 미납 카운트

### 화이트리스트

> [!important] 화이트리스트 ↔ 선수 자동 매칭
> 이메일 추가 시 선수도 같이 지정해두면 그 사람 첫 로그인 시 자동 매칭됩니다.

### 변경 이력 (감사 로그)

## 📚 데이터 모델 (참고)

```
dokkaebi/
├── seasons/{sYYYY}        시즌 (연도별)
├── players/{pid}          선수 + stats + seasonStats[sid]
│                          · isRegular · mainPosition · subPosition
│                          · photoURL · bio
├── matches/{mid}
│   ├── seasonId / opponent / date / location / type
│   ├── quarters[4]        쿼터별 lineup·events·formation·positions
│   ├── plannedSquads[4]   예정 스쿼드 (스쿼드 메이커)
│   ├── score/lineup/events  4쿼터 합산(표시용)
│   ├── momPlayerId · votingClosed
│   ├── rsvps/{uid}
│   └── votes/{uid}
├── photoPosts/{id}        사진 게시물 + likes + comments
├── announcements/{id}     공지
├── posts/{id}             자유게시판 글 + comments
├── allowedEmails/{...}    화이트리스트 (playerId 자동 매칭)
├── users/{uid}            로그인 사용자 + playerId
├── matchFees · monthlyDues 회비
└── auditLogs              변경 이력
```

## 🛠 자주 발생하는 상황

> [!question] 본인 선수가 안 보여요
> 마이페이지 → [연결] 버튼 → 직접 선택. 다음 로그인부터 자동 인식.

> [!question] MOM 투표가 안 눌려요
> 그 경기의 출전 명단에 포함된 사람만 가능. 안 뛰었으면 투표 불가.

> [!question] 시즌 드롭다운에 한 옵션만 있어요
> 관리자 대시보드 → "시즌 연도별 분리 실행"

> [!question] 상대팀이 같은데 다르게 표기됐어요
> 관리자 대시보드 → "상대팀명 통일 실행" (국대FC / FC국대 → 국대FC)
