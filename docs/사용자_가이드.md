📅 [3] 경기 페이지
━━━━━━━━━━━━━━━━━━━━

▶ 일반 사용자
▪ 4쿼터별 점수 / 라인업 / 포메이션 확인
▪ MOM 확인 (참여자만 투표 가능)
▪ 베스트 듀오 확인
▪ "결과 카드 이미지" 1080×1080 생성
   → 인스타 / 카톡 공유 OK

▶ 관리자
▪ 경기 등록·수정
▪ 결과 입력 (4쿼터 × 라인업 × 골/어시/카드)
▪ 스쿼드 메이커
   → 예정 경기에 4쿼터 라인업 미리 짜기
   → 단톡 공유 텍스트 자동 생성

💡 결과 입력 시 "예정 스쿼드 가져오기" 누르면
   미리 짠 라인업 한 번에 적용!

💡 [10] 자주 묻는 질문
━━━━━━━━━━━━━━━━━━━━

❓ 로그인했는데 본인 선수 자동 연결 안 돼요
→ 관리자에게 카톡 → 화이트리스트 등록 → 다시 로그인

❓ 프로필 사진 어떻게 바꿔요?
→ 마이페이지 → "🎨 샘플 선택" → 5탭 중 선택
   또는 "📁 직접 업로드"로 본인 사진

❓ 풋살 경기는 통계에 포함되나요?
→ 아니요!
   권용재 경기 / 풋살장 경기는 골득실·전적 제외

❓ 결과 카드 이미지는 어디서 만드나요?
→ 경기 상세 화면 하단
   "결과 카드 다운로드" 버튼 → 1080×1080 PNG

❓ MOM 투표는 누가 할 수 있나요?
→ 그 경기에 참여한 인원만 투표 가능

❓ 시즌은 어떻게 나뉘나요?
→ 년도별 (2025 / 2026 따로)
   "2025-2026" 식으로 묶지 않음

❓ 한글 초성 검색은 어디서 되나요?
→ 선수 목록 / 게시판 검색창
   "ㅎㅅ" → "호성" 같이 작동
