# 🐉 도깨비FC 앱 사용 가이드

여러분 모두 환영합니다! 도깨비FC 전용 웹사이트가 오픈됐어요.
아래 기능들을 활용해 팀 운영·기록·소통을 한 곳에서 해결합시다.

────────────────────
🌐 **접속**
────────────────────
주소: https://hoseong-lee.github.io/Dokkaebi_FC/

· 본인 구글 계정으로 로그인 (관리자가 화이트리스트에 등록한 이메일만 허용)
· 모바일에서 "홈 화면에 추가" 하면 앱처럼 사용 가능
· 화면 위 메뉴: 홈 · 경기 · 선수 · 랭킹 · 달력 · 공지 · 게시판 · 사진첩 · 더보기

────────────────────
⭐ **처음 한 번만: 내 선수 연결**
────────────────────
우측 상단 [내 선수 연결] 버튼 → 본인 이름 선택

→ 이후 MOM 투표, 본인 통계 표시에 정확히 반영됩니다.

────────────────────
🏠 **홈**
────────────────────
· 다가오는 다음 경기 한눈에
· 최근 결과 3건
· 시즌 공격포인트 TOP 3

────────────────────
📅 **경기**
────────────────────
**예정/완료 탭**으로 일정·결과 조회

각 경기 상세에서 가능한 것:
· 📍 경기장 위치 (지도 링크 누르면 길찾기)
· ✋ **참석 여부(RSVP)** — 참석/미정/불참 토글 (실시간 반영)
· 🏆 쿼터별 스코어·득점자·포메이션 도표
· ⭐ **MOM 투표** (경기에 참여한 선수만 가능)
· 📸 **결과 카드 만들기** — 인스타·단톡 공유용 1080×1080 이미지 자동 생성 후 다운로드/공유

**경기 일정의 양상 배지**:
· 🔥 대승(4점차+) / ✅ 승 / 🤝 무 / ❌ 패 / 💥 대패
· 1점 차 경기엔 🟣 "접전" 태그

────────────────────
🗓 **달력** (더보기 메뉴)
────────────────────
· 월간 캘린더에 경기 핀 표시 (승=파랑, 패=빨강, 무=회색, 예정=네이비)
· 셀 클릭 → 경기 상세

────────────────────
👥 **선수**
────────────────────
· 포지션별 4색 컬러 라벨 (🟡 GK · 🔵 DF · 🟢 MF · 🔴 FW)
· 선수 카드 클릭 → 본인 상세

**선수 상세에서**:
· 통산·시즌별(2025년/2026년) 통계 차트
· 시즌 선택 시 **월별 골/도움/출석 추이 그래프**
· **시즌별 비교표** (2025 vs 2026 + 통산, 추세 화살표)

────────────────────
🏆 **랭킹 · 기록**
────────────────────
· **팀 시즌 전적 카드**: 승-무-패 · 득실 · 승률 · 최근 5경기 폼
· **5개 탭**: 득점왕 · 도움왕 · 공격포인트 · 출석왕 · MOM
· 시즌 선택 가능 (통산/2025년/2026년)
· **🏆 시즌 베스트 11**: 가중치 점수로 4-3-3 자동 빌드
· **🏅 올해의 선수** 카드
· **상대팀별 전적(H2H)**: 🟢 우세 / ⚪ 백중 / 🔴 약세 자동 분류

────────────────────
🏆 **명예의 전당** (더보기)
────────────────────
시즌별 자동 시상 카드
· 👑 시즌 MVP · ⚽ 득점왕 · 🎯 도움왕 · 🏃 출석왕 · ⚡ 공격포인트왕

────────────────────
📢 **공지사항** (더보기)
────────────────────
· 관리자가 올린 팀 공지
· 📌 상단 고정 가능

────────────────────
💬 **자유게시판** (더보기)
────────────────────
· 누구나 글 작성 가능 (제목 + 내용)
· 댓글 작성/삭제
· 본인 글만 수정·삭제 (관리자는 전체)

────────────────────
📷 **사진첩** (더보기)
────────────────────
경기·일상 사진을 글 형태로 공유

작성 시:
· 제목 · 본문 · 태그(#국대FC #대승 등) · 관련 경기 · 다중 사진

조회 시:
· 태그·경기별 필터링
· 이미지 캐러셀(좌우 화살표)
· ❤️ **좋아요** (1인 1개)
· 💬 댓글

────────────────────
⚙️ **관리자 전용** (관리자만 보임)
────────────────────
**[관리자 → 대시보드]**
· 결과 미입력 경기 알림
· 이번 주 경기
· 초기 데이터 다시 가져오기

**[관리자 → 선수 관리]**
· 선수 등록/수정/삭제
· 🎨 **프로필 아바타** 샘플 12종 선택 또는 직접 사진 업로드
· 메인/서브 포지션, 등번호, ★ 고정멤버 여부

**[관리자 → 경기 등록]** 또는 경기 상세 → [스쿼드 메이커]
· 4쿼터별로 라인업·포메이션 별도 작성 가능
· ★ 고정멤버 전체 선택 한 번에
· + 용병 즉석 추가
· ⚡ **참석 인원 기반 포메이션 자동 추천**
· 인기 포메이션 칩 (4-3-3, 4-2-3-1, 4-4-2, 4-3-2-1) + 기타 6종
· 💬 **단톡 공유 텍스트** 한 번에 복사 (쿼터별 깔끔하게)

**[관리자 → 결과 입력]** (4쿼터)
· 쿼터 탭별 명단·스코어·득점자·어시스트·포메이션
· 📋 예정 스쿼드 → 쿼터로 가져오기
· ↻ 직전 쿼터 명단 복사
· MOM 직접 지정 또는 투표 결과로 마감

**[관리자 → 회비 관리]**
· 매치비: 경기별 출전자 납부 토글 + 일괄 처리
· 월회비: 월별 고정멤버 납부 토글
· 미납 카운트 자동

**[관리자 → 화이트리스트]**
· 로그인 허용 이메일 관리

**[관리자 → 변경 이력]**
· 누가 무엇을 언제 변경했는지

────────────────────
📌 **꿀팁**
────────────────────
1. 경기 끝나면 → 결과 입력 → MOM 투표 마감까지 한 번에
2. 사진/영상은 사진첩에 글로 공유 (태그 활용)
3. 스쿼드 짜고 [💬 단톡 공유] 누르면 카톡 붙여넣을 형식으로 자동 복사
4. 모바일에서 홈 화면에 추가 (PWA처럼 사용)
5. 본인 선수 연결을 안 하면 MOM 투표가 안 됩니다
