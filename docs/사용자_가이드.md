# 🐉 도깨비FC 앱 사용 가이드

안녕하세요 여러분!
도깨비FC 전용 웹사이트가 오픈됐고 기능이 풍성해졌어요.
운영·기록·소통을 한 곳에서 해결합니다.

━━━━━━━━━━━━━━━━━
🌐 **접속**
━━━━━━━━━━━━━━━━━
주소: https://hoseong-lee.github.io/Dokkaebi_FC/

· 본인 구글 계정으로 로그인 (관리자가 사전 등록한 이메일만 허용)
· 모바일에서 "홈 화면에 추가" 하면 앱처럼 사용 가능
· 상단/하단 메뉴: 홈 · 경기 · 선수 · 랭킹 · 달력 · 공지 · 게시판 · 사진첩 · 더보기

━━━━━━━━━━━━━━━━━
👤 **첫 접속 후 마이페이지부터**
━━━━━━━━━━━━━━━━━
헤더 우측 **[★ 내 정보]** 클릭

· **연결된 선수** 확인 (자동 매칭됐을 가능성 큼, 안 됐으면 직접 [연결])
· **프로필 사진** — 도깨비FC 유니폼 캐릭터 24종 (남12 + 여12, 다양한 피부톤·헤어컬러) 또는 본인 사진 업로드
· **한마디** 입력 (80자, 예: "슈팅보다 도움 더 좋아하는 윙백")
· **내 시즌 통계** + **단짝 콤비** + **획득 뱃지** 한눈에

━━━━━━━━━━━━━━━━━
🏠 **홈**
━━━━━━━━━━━━━━━━━
· 다가오는 다음 경기 + D-Day 배지
· 최근 결과 3건
· 시즌 공격포인트 TOP 3

━━━━━━━━━━━━━━━━━
📅 **경기**
━━━━━━━━━━━━━━━━━
**예정/완료 탭**

각 경기 상세에서:
· 📍 경기장 위치 (지도 링크)
· ✋ **참석(RSVP)** — 참석/미정/불참 실시간 토글
· 🏆 쿼터별 스코어·득점자·포메이션 도표
· ⭐ **MOM 투표** (그 경기 참여 선수만 가능)
· 📸 **결과 카드** — 인스타·단톡 공유용 1080×1080 이미지 자동 생성

**경기 양상 배지** (스코어 차로 자동):
🔥 대승(4점차+) · ✅ 승 · 🤝 무 · ❌ 패 · 💥 대패
+ 1점 차이면 🟣 접전 태그

**📋 예정 스쿼드 = 4쿼터 각각** 미리 짤 수 있고, 결과 입력 시 한 번에 가져오기 가능.

━━━━━━━━━━━━━━━━━

· 🔵 예정 · 🟦 승 · 🤍 무 · 🟥 패
· 클릭 → 경기 상세

· 시즌 선택 시 월별 골/도움/출석 추이 그래프
· 시즌별 비교표 (2025 vs 2026 + 통산, ↑↓ 추세)
· 🏅 획득 뱃지

🔍 **검색 꿀팁**
━━━━━━━━━━━━━━━━━
스쿼드/결과 입력의 선수 검색은 **초성도 매칭**:
· `ㅇㅎㅊ` → 유희창
· `ㄱㅅㅇ` → 김산이
· 등번호(11) · 포지션(CB) 도 검색 가능

━━━━━━━━━━━━━━━━━
⚙️ **관리자 전용**
━━━━━━━━━━━━━━━━━
**대시보드** — 결과 미입력 알림 / 이번 주 경기 / 초기 데이터 가져오기 / 시즌·상대팀 정리 자동화

**선수 관리** — 등록/수정/삭제 · 🎨 프로필 아바타 24종 또는 직접 업로드 · 메인/서브 포지션 · 등번호 · ★ 고정멤버 여부

**경기 등록/수정 또는 [스쿼드 메이커]**
· 4쿼터별 라인업·포메이션 별도 작성
· ★ 고정 전체 1클릭 / + 용병 즉석 추가
· ⚡ **참석 인원 기반 포메이션 자동 추천**
· 인기 포메이션 칩 (4-3-3 / 4-2-3-1 / 4-4-2 / 4-3-2-1)
· **선수별 쿼터 카운트 배지** (1쿼만=주황, 2~3쿼=초록, 풀쿼=골드) — 회전 균형 한눈에
· 💬 **단톡 공유 텍스트** 1클릭 복사 (쿼터별 + 포지션별 깔끔 정렬)

**결과 입력** (4쿼터 탭)
· 📋 예정 스쿼드 가져오기 / ↻ 직전 쿼터 명단 복사

**회비 관리** — 매치비/월회비 토글 + 미납 카운트

**화이트리스트** — 이메일 추가 시 **선수 연결**까지 같이 → 그 사람 첫 로그인 시 자동 매칭

**변경 이력** — 감사 로그

━━━━━━━━━━━━━━━━━
📌 **꿀팁**
━━━━━━━━━━━━━━━━━
1. 경기 끝나면 → 결과 입력 → MOM 투표 마감
2. 마이페이지에서 본인 사진/한마디 꼭 채우기
3. 스쿼드 짜고 [💬 단톡 공유] 누르면 카톡 붙여넣기 가능
4. 모바일은 홈 화면 추가로 앱처럼 사용
5. 본인 선수 연결 안 하면 MOM 투표 불가
